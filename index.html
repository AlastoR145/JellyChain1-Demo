ProgPOW Blockchain

JellyChain1 is an experimental programmatic blockchain network built to scale out database software and other applications. JellyChain1 is created to save energy when mining transactions on POW blockchains, and also to help fight inflation in the financial system.

The Problem

Most Proof-Of-Work blockchains rely on expensive GPUs to validate a transaction on the blockchain. These GPUs are often very loud and costly on the energy bills. These upfront cost make it harder for miners to profit and causes razor thin margins for miners.

The solution

JellyChain1 believe we are the answer to solve the above problem. JellyChain1 is an automative ProgPOW blockchain. We have automated the mining process with our consensus layer. This layer listen to the main chain for updates or changes. So, when a transaction is sent to the blockchain it will be automatically processed. This is what makes JellyChain1 trustless. Because anyone can just download a copy of JellyChain1 from the GitHub repository, and run the chain without needing expensive GPUs.
